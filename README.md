# Venue Rental ‚Äì Backend

This repository contains the **backend** for the Venue Rental application. It provides APIs and business logic to support user authentication, venue listings, bookings, and email notifications.

The backend is designed to be reliable, scalable, and easy to integrate with the frontend.

## Features
- User authentication and authorization
- Venue listing and management
- Secure booking system
- Email confirmations and booking reminders
- User dashboard data handling
- RESTful API architecture

- Serve secure API endpoints to the frontend
